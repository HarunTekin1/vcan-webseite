So siehst du deine lokale Webseite (einfach):

1) Diese Datei lesen.
2) Doppelklick auf open-site.bat im gleichen Ordner -> Seite öffnet im Browser.

Falls das nicht geht:
- Rechtsklick auf index.html -> Öffnen mit -> Deinen Browser wählen.

Optional (nicht nötig, nur falls jemand hilft):
PowerShell hier öffnen und eingeben:
python -m http.server 5500
Dann Browser: http://localhost:5500/

Fertig. Du brauchst nichts zu installieren, um nur anzuschauen.
